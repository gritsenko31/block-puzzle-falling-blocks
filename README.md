# Block Puzzle: Falling Blocks

A classic block puzzle game built with React Native (Expo).

## 🎮 About

Block Puzzle: Falling Blocks is a casual puzzle game where colorful blocks fall from the top of the screen. Your goal is to arrange them to complete full lines and score as many points as possible.
